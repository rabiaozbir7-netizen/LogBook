# Akıllı Günlük & Egzersiz Asistanı 🧠✨

Ruh haline göre şekillenen, dinamik renk ve yapay zeka destekli öneriler sunan, geçmişteki "MUTLU" anılarını zor günlerinde karşına çıkaran duygu farkındalığına sahip benzersiz bir akıllı günlük uygulaması.

## 🌟 Temel Özellikler

- **Duygu Odaklı Dinamik Temalar:** "Mutlu", "Enerjik", "Sakin", "Yorgun", "Stresli" ve "Kötü" ruh hallerine göre eş zamanlı değişen arka plan renkleri ve asistan tavsiyeleri.
- **Geçmişten Gelen Destek (Dost Hafızası):** Kötü veya Stresli hissettiğin bir günde uygulamaya girdiğinde, daha önce kaydettiğin en güncel mutlu anın (ses kaydı veya metin) karşına çıkarak sana destek olur.
- **Kendime Notlar & Sesli Günlük:** Her güne dair kısa notlar alabilir veya gününü mikrofona sesli şekilde anlatarak bir anı defteri oluşturabilirsin.
- **Düzenli Kayıt Kütüphanesi:** Kaydettiğin tüm sesler ve metinler "Kayıtlarım" sekmesinde, o an hissettiğin duygunun renkleriyle depolanır.
- **Yerel ve Kalıcı Hafıza (Offline Native):** Uygulama AsyncStorage aracılığıyla tüm notlarını ve ses kayıtlarını internet olmadan saklar.

## 🛠 Kullanılan Teknolojiler

- **[React Native] & [Expo GO]**
- **Sistem İçi Ses İşleyici:** `expo-av`
- **Navigasyon ve Sekmeler:** `@react-navigation/native` & `@react-navigation/bottom-tabs`
- **İkon Mimarisi:** `lucide-react-native`
- **Lokal Depolama:** `@react-native-async-storage/async-storage`

## 🚀 Kurulum (Nasıl Çalıştırabilirim?)

Bu projeyi bilgisayarında çalıştırmak istiyorsan aşağıdaki adımları sırayla takip edebilirsin:

### Gereksinimler
- Bilgisayarında [Node.js](https://nodejs.org/) kurulu olmalı.
- Telefonunda (App Store veya Google Play'den) **"Expo Go"** uygulaması yüklü olmalı.

### Adımlar

1. **Projeyi Klonlayın veya İndirin:**
   ```bash
   git clone <GTIHUB_REPO_LINKI>
   cd day
   ```

2. **Gerekli Paketleri Yükleyin:**
   ```bash
   npm install
   ```

3. **Uygulamayı Başlatın:**
   ```bash
   npx expo start
   ```

4. **Telefonda Önizleme Yapın:**
   Ekrana gelen QR kodunu akıllı telefonunun kamerasıyla (ya da Expo Go uygulamasından) okut. Proje anında telefonunda çalışmaya başlayacaktır!
